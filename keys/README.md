# Alipay sandbox keys

Put your Alipay sandbox key files in this directory:

- `alipay_app_private_key.pem`: your application private key generated by the Alipay key tool.
- `alipay_public_key.pem`: the Alipay public key copied from the sandbox console.

Do not put the application public key here. The application public key is only uploaded to the Alipay sandbox console.

These key files are ignored by Git.
